⬡ Interview Assistant

Windows setup — do this once, takes 2 minutes
1
Enable Stereo Mix (captures everything playing through your speakers)

Right-click the 🔊 speaker icon in taskbar → Sound settings → More sound settings

Go to the Recording tab → right-click empty area → tick "Show Disabled Devices"
Recording tab shows:
☑ Microphone — enabled
☐ Stereo Mix — right-click → Enable → right-click again → Set as Default Device
2
Set Stereo Mix as default in Chrome

In Chrome address bar → chrome://settings/content/microphone
Change the default microphone to Stereo Mix
Default microphone: [Stereo Mix (Realtek …) ▼]

If Stereo Mix is not available → install free VB-Cable:
vb-audio.com/Cable → install → reboot → use CABLE Output as mic, set CABLE Input as your audio output device in Windows
3
Get a free Groq API key
Go to console.groq.com → Sign up → API Keys → Create Key
Paste it in the sidebar and click Save
4
Host on GitHub Pages (so Chrome remembers permission forever — no more popups)

Push index.html to a GitHub repo → Settings → Pages → Deploy
Open https://yourname.github.io/interview-assistant — done!

OR just open this file locally — Chrome will ask once per session
💡 Why Stereo Mix works: It's a Windows feature that treats your speaker output as a microphone input. When the interviewer speaks in Google Meet, it plays through your speakers — Stereo Mix captures that and sends it to this app, which transcribes it with Groq Whisper and generates an answer.
